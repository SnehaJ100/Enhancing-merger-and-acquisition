# Enhancing-merger-and-acquisition-prediction-through-deep-learning-algorithms

## M&A Target Prediction — Deep Learning vs. Classical ML

A comparative study predicting merger and acquisition (M&A) targets using a 
Feedforward Neural Network benchmarked against 7 classical machine learning models.

### Results

- Best model: Feedforward Neural Network — **80% accuracy**
- Models benchmarked: Naïve Bayes, KNN, Decision Tree, Logistic Regression, 
  Random Forest, Gradient Boosting, AdaBoost

### Dataset

10,000+ companies with financial features; sourced and preprocessed for 
binary classification (M&A target: yes/no).

### Tech Stack

Python, Scikit-learn, Pandas, Jupyter Notebook

### Project Structure

- `merger_prediction_model.ipynb` — full pipeline (EDA, preprocessing, 
  model training, evaluation)
- `Data/` — input dataset
- `Outputs/` — saved model outputs and evaluation charts
